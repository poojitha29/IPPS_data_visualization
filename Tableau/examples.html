DRG Definition for Provider State

Description-

Various DRGs are used in hospitals throughout the nation for different medical procedures. This webpage represents Top 100 DRG Definition used in hospitals throughout the States.

Instructions on how to interpret the Map
  1. In the top right corner of the map is a filter named “DRG Definition” that contains all the top 100 DRGs in the form of a Dropdown List.
  2. On selecting a DRG all the states that have hospitals using that DRG will get highlighted. For example, select first DRG DRG-039, you will notice that some states in the map have become blue.
  3. Those are the states that have hospitals, which provide DRG DRG-039.
  4. Try hovering on the Map, it will show the names of the states.
  5. Now select DRG, DRG-069; notice something??
  6. The color of states has changed to light Green; notice something else?
  7. Try comparing the states that got highlighted when you selected drug DRG-039 with the ones that got highlighted when you selected DRG DRG-069
  8. There is one state that does not get highlighted when you select DRG-069.
  9. Which one is it?
  10. The answer is Wyoming (WY).

Note: You might not want to go to Wyoming if you are seeking a procedure that requires DRG-069. If you want to know what hospitals use these DRGs in each state got to Tab “State Wise Breakdown of DRG Definition”

State Wise Breakdown of DRG Definition

Description-

Each state has hospitals that uses all of the DRGs. This web page represents a map that includes all the hospitals in a particular state offering all the DRGs.

Instructions on how to interpret the Map
  1. To select a particular State go to the top right corner of the Map and select the state of your choice by using the drop down list named “Provider State”.
  2. After selecting a state, the map will be populated with dots representing DRGs and hospitals providing that DRG.
  3. Color Code of each DRG is described in the form of a list under the filter “Provider State”.
  4. On clicking on a particular DRG the map will highlight only the selected DRG
  5. Hover on the dots in the Map to see Hospital Names and Zip Codes.
  6. To select any other DRG just click on it.
  7. To see all the DRGs Double click on the DRG currently selected or in other words deselect the DRG currently highlighted in the list.

Note: If you want to know which state is more expensive for a DRG go to next tab “Trend of Average Payments and Total Discharges”

DRG Definition vs Total Discharges

Description-

This webpage contains a table that represents number of discharges per hospital for all the Drugs i.e. row 1 contains name of the hospitals and state that hospital belong to and column 1 contains Drug Names.

Instructions on how to interpret the Map
  1. Select a Drug by using the Drop Down list on top right corner named “DRG Definition”. For example- Select first Drug DRG-039.
  2. Look at the filter named “Sum” on top right corner of the chart, under “DRG Definition”. The Sum filter represents a range of value from min to max number of discharges, for example, for Drug DRG-039, minimum number of discharges is 11 and maximum is 190, where values below the average are represented by red and values above the average are represented by green
  3. Try to find the hospital with maximum number of discharges (look for darkest green color with label 190), in our case it is “Kansas Heart Hospital”.
  4. Result – For Drug DRG-039 Kansas Heart hospital located in Kansas has maximum number of discharges
  5. Similarly try for other values of Drugs, you can also filter by states.

DRG Definition vs Average Covered Charges

Description-

Each state has hospitals that uses all of the Drugs. This web page represents a map that includes all the hospitals in a particular state offering all the Drugs.

Instructions on how to interpret the Map
  1. To select a particular State go to the top right corner of the Map and select the state of your choice by using the drop down list named “Provider State”.
  2. After selecting a state, the map will be populated with dots representing Drugs and hospitals providing that Drug.
  3. Color Code of each Drug is described in the form of a list under the filter “Provider State”.
  4. On clicking on a particular Drug the map will highlight only the selected Drug
  5. Hover on the dots in the Map to see Hospital Names and Zip Codes.
  6. To select any other Drug just click on it.
  7. To see all the Drugs Double click on the Drug currently selected or in other words deselect the Drug currently highlighted in the list.

Note: If you want to know which state is more expensive for a Drug go to next tab “Trend of Average Payments and Total Discharges”
